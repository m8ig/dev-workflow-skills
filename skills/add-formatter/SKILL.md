---
name: add formatter
description: Use when setting up code formatting for a new project. Covers Prettier installation and the standard .prettierrc config used across projects.
---

Install Prettier with the following config

.prettierrc

```json
{
    "bracketSameLine": true,
    "bracketSpacing": true,
    "htmlWhitespaceSensitivity": "ignore",
    "printWidth": 120,
    "singleQuote": true,
    "tabWidth": 4,
    "trailingComma": "all",
    "singleAttributePerLine": true
}
```
